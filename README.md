# Riichi Mahjong Forge

Riichi Mahjong Forge adds a fully playable Riichi Mahjong table to Minecraft. Play with up to three other players (or simple bots) into classic Riichi Mahjong.

## Features

- Full Riichi Mahjong ruleset with all yaku and specific rules (even chankan)
- 4 players — bots take empty seats
- Advancement for getting each yaku
- Scoring by han/fu with payment splits (dealer/non-dealer, tsumo/ron)
- EMI integration for custom recipes
- (spoilers...) crafting
- (spoilers...) generator
